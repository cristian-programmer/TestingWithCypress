# TestingWithCypress
This repository is for end-to-end testing of a web page.
<br />
<br />
<br />

## Build
- clone repository 
```sh
    git clone https://github.com/cristian-programmer/TestingWithCypress.git 
 ```
 - enter to folder
 ```sh
    cd TestingWithCypress
 ```
  - install dependencies
 ```sh
    npm install
 ```

   - run test
 ```sh
    npm run cypress:open
 ```
